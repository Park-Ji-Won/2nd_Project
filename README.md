# 객체지향응용프로그래밍 2차 프로젝트 발표

### 학생 성적 관리 프로그램

각 학생과 사용자인 교육자의 정보를 입력 및 정정 그리고 확인

입력된 학생의 정보에 고안된 함수를 통해 과목명과 점수를 등록

입력된 과목과 점수를 바탕으로 평균 구하는 것을 시작으로

그 학생들이 속한 반 전체의 평균과 최고점 최저점 그리고 학생들의 순위까지

#### 개선점

---

여기에 데이터베이스를 접목해 정보를 자동적으로 입력하거나 

미리 외부에 데이터를 저장해 필요할때 불러오는 기능등을 구현하고 

사용하기 좋게 GUI를 추가한다면 좀더 실용적인 프로그램이 될것이라 본다.
